## @thoth in Chat

Type `@thoth` in the VS Code Chat panel to ask computational questions inline:

```
@thoth What is the binding energy per nucleon of iron-56?
```

Results are streamed as markdown directly in the chat. For simulations and interactive visualizations, click "View in Thoth Panel" to open the full UI.

Thoth also registers tools that other chat participants can use for computation and code execution.
