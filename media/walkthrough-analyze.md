## Analyze Code

Thoth Alpha can analyze code directly from your editor:

- **CodeLens:** Look for "Analyze with Thoth" above functions and classes
- **Context Menu:** Select code, right-click, choose **Thoth Alpha > Analyze Selection**
- **Keyboard:** Select code and press `Cmd+Shift+E` (Mac) or `Ctrl+Shift+E` (Windows/Linux)

Thoth will explain the algorithm, suggest optimizations, and identify potential issues.
