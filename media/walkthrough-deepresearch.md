## Deep Research Mode

Deep Research uses Google Gemini to perform exhaustive, multi-source analysis. It takes 30-60 seconds but returns comprehensive results.

**Setup:**
1. Get a Gemini API key at [Google AI Studio](https://aistudio.google.com/apikey)
2. Run **Thoth Alpha: Set Gemini API Key** from the command palette
3. Click the "Deep Research" button in the search panel

Your API key is stored securely in your OS keychain.
